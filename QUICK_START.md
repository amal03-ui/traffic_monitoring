# 🚀 Quick Start Guide

## For Your Friends - Super Easy Setup!

### **Step 1: Get the Code**
```bash
git clone https://github.com/aromal0708/Advanced-Driver-Assistance-System.git
cd AdvancedTrafficMonitoringSystem
```

### **Step 2: One-Click Setup**
Double-click on:
- **`setup.bat`** (if you use Command Prompt)
- **`setup.ps1`** (if you use PowerShell)

This will automatically:
- ✅ Create virtual environment
- ✅ Install all dependencies
- ✅ Download AI model
- ✅ Set up everything for you!

### **Step 3: Start the App**
Double-click on:
- **`start_server.bat`** (if you use Command Prompt)
- **`start_server.ps1`** (if you use PowerShell)

### **Step 4: Open Browser**
Go to: **`http://localhost:5000`**

## 🎉 That's it! Your traffic monitoring system is running!

---

## 🆘 Having Problems?

1. **Run system check**: Double-click `check_requirements.bat`
2. **Make sure you have**:
   - Python 3.10+ installed
   - Internet connection (for downloading AI model)
   - Administrator permissions (if needed)

## 📱 How to Use

1. **Upload traffic images** using the web interface
2. **See vehicle detection** in real-time
3. **Watch traffic signals** change automatically
4. **View statistics** and processing times

---

## 🛑 To Stop the Server
Press `Ctrl + C` in the terminal window
